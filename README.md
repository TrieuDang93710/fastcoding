## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Web front-end

```bash
# cd into web folder
$ cd web-front-end

# run live-server on brownser
$ Right click -> Open with live server(Alt + L, Alt + O)
```
